🔄 Cómo funciona la sincronización

Cuando guardas cualquier dato (nota, lab, nuevo paciente), se sube al Worker inmediatamente.
Cada 30 segundos, el app consulta al Worker si hubo cambios de otro usuario y actualiza automáticamente.
Si no hay internet, trabaja en modo offline con caché local hasta recuperar conexión.

📋 Setup del Worker v2 (si no lo has hecho)

  1. Cloudflare Dashboard → Workers & Pages → Create Worker → pega cloudflare-worker-v2.js
  2. Workers & Pages → KV → Create namespace → Nombre: PISOLIBRO_KV
  3. Tu Worker → Settings → Bindings → Add: Variable KV → namespace PISOLIBRO_KV
  4. Settings → Variables → Add: ANTHROPIC_API_KEY (Encrypt ✓) y ACCESS_TOKEN (Encrypt ✓)
  5. Deploy → copia URL → pega arriba
